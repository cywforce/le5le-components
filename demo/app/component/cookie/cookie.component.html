使用方式

getCookie(name:string)：
[静态方法]读取名为name的cookie。


setCookie(name:string, value:string, expires?:number, path?:string, domain?:string)：
[静态方法]设置名为name的cookie。
value - cookie值
expires - cookie有效期的毫秒数
path - cookie所在的目录
domain - cookie域名


deleteCookie(name:string, value:string, expires?:number, path?:string, domain?:string)：
[静态方法]删除名为name的cookie。
expires - cookie有效期的毫秒数
path - cookie所在的目录
domain - cookie域名
